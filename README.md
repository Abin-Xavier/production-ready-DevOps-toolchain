# ☁️ CloudStream Dashboard - Enterprise DevOps Toolchain

An automated, containerized web application managed through a modern GitOps CI/CD pipeline. This project demonstrates how to offload high-compute compilation workflows to the cloud, ensuring standardized builds across engineering teams.

## 🚀 Key Features
* **Containerized Architecture:** Packed via Docker to eliminate local virtualization system discrepancies.
* **Automated CI/CD Pipeline:** Powered by GitHub Actions to automatically lint, build, and version control artifacts on every code check-in.
* **Cloud Native Workflow:** Completely decoupled from local hardware constraints.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Containerization:** Docker
* **CI/CD Orchestration:** GitHub Actions
* **Artifact Registry:** Docker Hub
